== Description ==

This Plugin is useful to order or reorder default blog category and custom post type taxonomy and terms, plugin is ready to use no configuration required. User friendly interface make your job done easily. plugin support multiple taxonomies from same post type as well.

== Features ==

+ Useful for post category ordering
+ Useful for custom post taxonomy ordering
+ Easy to use interface
+ No skill required to use this plugin.
+ No integration required.
+ Works With any WordPress theme

== Installation ==

upload terms-order folder to wp-content/plugins directory and activate plugin from wp-admin -> plugins
now you can reorder any taxonomy by going to particular post/custom post type -> click terms order menu -> select a taxonomy if select post type has mulitple taxonomies -> drag and drop terms up/down and hit update button
